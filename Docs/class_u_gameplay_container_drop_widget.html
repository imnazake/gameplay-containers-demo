Detailed Description

UGameplayContainerDropWidget

A widget used for drag and drop operations that are meant to take the items out of a gameplay container and spawn them in the game world if your game allows this.

This is usually placed in the background or in specific areas on the screen where you want the dragged item to be dropped in order to remove it from the owning gameplay container and spawn it in the game world.

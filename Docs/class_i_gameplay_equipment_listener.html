Detailed Description

IGameplayEquipmentListener

This interface defines a contract for gameplay container components that intend to observe and respond to changes in gameplay equipment manager within a container. Classes implementing this interface are required to provide implementations for functions, which serve as callbacks for when equipped items are modified within the gameplay equipment manager.

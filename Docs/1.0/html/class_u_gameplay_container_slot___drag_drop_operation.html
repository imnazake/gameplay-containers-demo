Detailed Description

UGameplayContainerSlot_DragDropOperation

A custom drag-and-drop operation for gameplay container slots.

This class represents a custom Drag-and-Drop operation specifically designed for use with gameplay containers. It extends UDragDropOperation and provides additional functionality to facilitate interactions with gameplay container slots.

Detailed Description

FGameplayContainerStartupItem

Represents a startup item configuration for gameplay containers.

This struct defines a startup item that can be added to a gameplay container when it is initialized. It contains information about the item's slot type, item definition, quantity, and whether it's forced.

The properties of this struct can be edited anywhere and are readable in Blueprints.

Note
This struct is primarily used for defining the initial contents of a gameplay container when it is initialized, allowing you to specify what items should be present in the container at the start.

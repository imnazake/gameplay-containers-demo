Detailed Description

FGameplayContainerOverflowingItem

Represents an overflowing item that cannot fit into a gameplay container's slots.

This structure stores information about an item that exceeds the available space in the container's slots. It includes the item instance, slot type, quantity, and a reference to the owning container component.

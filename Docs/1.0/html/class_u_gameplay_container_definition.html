Detailed Description

UGameplayContainerDefinition

A base class for defining gameplay container definitions.

This class serves as a base class for creating gameplay container definitions. ContainerManager definitions define the structure and behavior of gameplay containers. They can contain fragments and specify usage-blocking tags to control how containers are used in the game.

Detailed Description

UGameplayContainerComponent

This component manages and interacts with gameplay containers, enabling the creation, modification, and querying of slots to hold and manage items or objects.

Containers can be created dynamically, and slots within them can be added or removed, offering flexible and customizable storage for gameplay objects.

Note
This class is intended to be subclassed and extended to implement specific container behaviors.
See also
